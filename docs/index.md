### As Easy as 1, 2, 3

```bash
# clone the project
git clone git@github.com:funnyzak/vuepress-starter.git && cd vuepress-starter

# install dependency
yarn

# start writing
yarn docs:dev

# build to static files
yarn docs:build
```
